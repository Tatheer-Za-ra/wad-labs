What I did and Why I did:

For Table

To enhance readability and organization, I used a table element with multiple attributes.The table structure consists of sections like thead, tbody, and tfoot to divide headers, body content, and footer information, as well as a caption for the title and a colgroup for column styling. For clarity, the second header row (#secheader) uses colspan to merge columns. Table styling is applied by the CSS, which also sets border-collapse: collapse; to merge borders, adds padding for spacing, and uses different background colors to distinguish different sections. To improve interactivity, the hover effect (tr:hover) modifies the color of the row. Specific IDs such as #cgpa and #srno, along with the .gpa class, guarantee accurate column width and background highlights for GPA related data.Moreover,Footer aligns text and Maintains a consistent theme.

For From

The HTML form is designed to gather information about student grades, including contact details, personal information, semester specifics, and feedback. To improve user interaction, it has multiple input types, including:

  • text fields
  • radio buttons
  • checkboxes
  • dropdown menus
  • range slider

To guarantee a neat layout, the form is organized using the fieldset and div sections. Additionally, by directing accurate data entry, placeholders and input validation patterns enhance the user experience. Background colors, padding, and hover effects are used which keeps the user interface pleasing.
